# 长任务提示词模板 v1

> **用途**: 给任何一个接手的 agent 快速定位项目、不踩旧坑、直接开工。
> **使用方式**: 复制此模板，替换 `[方括号]` 内容，保留 "不要做的事" / "结论边界模板" / "框架约束" 三部分长期复用。

---

这轮任务在总地图中的位置：

   - Stabilization
   - Organization
3. hallucination 与 sycophancy 可能是不同 controllability object
4. 任务目标是补 proof obligation，不是再造一个"功能 demo"
5. 若本轮结果只支持扰动敏感，不要升级成方向因果证明

## 使用说明

每次新任务只需修改这 **5 个区域**：

1. **任务定位**
2. **先读文件**
3. **当前可信状态**
4. **具体任务**
5. **验收标准**

以下部分 **长期复用**，除非项目换代：

- 不要做的事（微调即可）
- 结论边界模板
- 框架约束
