### 3.3 L3 的"负效应"

样本 14 和 24 中，L3 combined 消融产生的是**负**Δ：
- Sample 14: L3 = **-0.008**（消融 L3 后幻觉偏好反而略微增强）

## 4. 解释：完整的信息流模型

### 4.1 三种测量的统一

```
                  L0          L1       L2        L3
                  │           │        │         │
 Embedding ──────→[attn]─────→[...]───→[...]────→[attn]
  "funding"       [ffn ]                        [ffn ]
                  └─┬─┘                        └─┬─┘
                    │                            │
    全层消融L0: kill 全输出                     全层消融L3: kill 全输出
    Δ≈? (未单独测)                              Δ=+0.41 ← 截断了L0→L3的累积残差!
                    │                            │
    双模块L0: kill attn+ffn                     双模块L3: kill attn+ffn  
    Δ=+0.339 ← 真正的信息入口!                  Δ=+0.008 ← L3本身没有新信息
```

### 4.2 为什么全层 L3 看起来有峰值？

全层消融在 L3 看到 +0.41，因为它一次性清空了 L3 的完整输出——这个输出中包含了：
- L0 的 attn+ffn 贡献（经过 L1-L2 残差传播）
- L1 的 attn+ffn 贡献
- L2 的 attn+ffn 贡献
- L3 本身的 attn+ffn 贡献

全层测量的是**累积残差的终点强度**，不是**信息进入系统的位置**。

### 4.3 完整的信息处理管道

```
Token "funding" 首次出现 (pos=7):

L0:   嵌入→表示转换   ←── 信息入口 (Δ=+0.34)
      attn+ffn 首次处理 "funding"

L1-2: 信息传播        ←── 信息沿着残差流扩散
      低层处理，连续整合

L3:   信息积累点      ←── 累积残差在此处最丰富
      全层截断 Δ=+0.41，但净贡献仅 +0.008

L4-15: 语义丰富化     ←── 深层语义处理

L16:  次高峰           ←── FFN 知识检索
      可能对应幻觉生成的知识检索阶段
```

## 5. 反幻觉干预策略修正

### 5.1 之前（P26-P28）

```
干预 L3 → 全层消融看到峰值
```

### 5.2 现在（P30 确认）

```
干预 L0 → 真正的信息入口
         L0 抹除 "funding" 的首现处理 → Δ=+0.34
         ≡ 消除 83% 的累积 L3 效应 (0.34/0.41 = 83%)
```

**L0 单层干预即可消除 83% 的幻觉因果信号。**

### 5.3 强化方案

```
L0 双模块消融 (attn+ffn):  Δ=+0.34  (单层即可)
L0 + L16 双消融:           Δ≈0.34+0.06 = 0.40  (接近全层 L3 效果)
```

L16 作为次高峰（可能对应知识检索阶段），提供了额外的 6% 增益。结合 L0(入口)+L16(检索) 可达到接近全层 L3 的效果。

## 6. P26→P30 知识链条总结

```
P26:  "funding" 全层峰值在 L3 (Δ=+0.41)    → 发现因果关系
P27:  不同 token 峰值不同 (L3/L4)           → L3 不通用
P28:  首现必在 L3，次现衰减 78%             → first-mention effect
P29:  拆分子模块→L3 消失, L0 成入口          → L3 是组合效应
P30:  双模块联合消融确认 L0 40x 碾压 L3     → 最优干预层确定
```

**最终模型：**

> **首现关键因果 token 的信息在 L0（第一层）通过 attn+ffn 联合首次被处理。**
> **这个信息沿 L1-L3 的残差流传播并积累，在 L3 累积到最大强度。**
> **但信息在 L0 进入系统时就是完整的——L1-L3 只是传播，不是重新编码。**
> **反幻觉干预的最优位置是 L0（单层可消除 83% 的因果效应）。**
