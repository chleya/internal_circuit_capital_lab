## 二、外部分析输入（五条理论主线）

### 主线 5：几何拓扑 / 流形保护
- **来源**：外部 LLM #5
- **核心论点**：
  - 决策边界是高曲率流形，非平坦超平面——线性 steering =「切线失败」
  - 均值差向量是密集特征（语调+句法+逻辑+噪声混合体）
  - 「正确性」是计算而非状态——修改表征会破坏中间变量
  - 模型有「世界模型」（知道真相）和「策略头」（过滤真相取悦用户）
  - **流形保护实验**：冻结语法/语义 Head，仅介入谄媚维度
- **建议**：PCA 子空间选择性 REPLACE（已执行为 M7-G）

### 主线 1：Polysemanticity / Epiphenomenon 假说
- **来源**：外部 LLM #1
- **核心论点**：均值差向量是密集向量（含句法、位置、路由混合特征），放大时摧毁模型能力；探针捕获早期决策的「废气」
- **建议**：Rimsky 式 token 级决策定时 + activation patching

### 主线 2：VAE 架构启示（Qwen-Image-VAE 2.0）
- **来源**：外部 LLM #2
- **核心论点**：
  - GSC（全局跳跃连接）→ 「因果直通」保护离散结构
  - 语义对齐 → 将 Observation 转化为 Operational 效率
  - 非对称架构 → 检测 vs 生成所需计算资本不对等
  - **«扩维而非偏移»** → 通过增加 latent 通道而非在原有维度上偏移来解决问题
- **建议**：LoRA 式旁路适配器 + 探针门控

### 主线 3：Flan 2.0 资本阈值理论
- **来源**：外部 LLM #3
- **核心论点**：
  - CoT 需要 Circuit Capital，0.5B 参数密度不足
  - 小模型没有独立的验证回路——「正确性」是涌现结果，不是独立维度
  - 对抗性指令资本库（Antagonistic Instruction Capital）→ 微调注入验证回路
- **建议**：Data Mixture（70%标准 + 20%事实校验 + 10%偏好对抗）+ 离散翻转测试

### 主线 4：三机制诊断
- **来源**：外部 LLM #4
- **核心论点**：
  - 带宽瓶颈：896D 残差流过度拥挤，非正交特征叠加
  - 回路匮乏：同一 Head 承担多任务，无助产头
  - 吸引子坍塌：RLHF 训练使谄媚成为低能耗捷径
- **建议**：Path Patching → 找到受挤兑最严重的多义性 Head

## 三、关键实验发现（里程碑）

### 发现的等级

| 等级 | 发现 | 实验 |
|---|---|---|
| 🥇 | **REPLACE 工作、ADD 失败**——谄媚不是单方向，需保留多维 pattern | PA1, PA2 |
| 🥇 | **类内方差是根因**——ADD保留每个样本的独特噪声，REPLACE消除之 | M7-C |
| 🥈 | **L18-L22 是关键控制窗口**——替换这 5 层可实现 100% 翻转 | PA2 |
| 🥈 | **谄媚是预训练先验**——Base 和 Instruct 均为 100% 谄媚 | M7-D |
| 🥉 | **MLP > Attention**——谄媚信号更多在 FFN 计算中，而非注意力路由 | M7-A |
| 🥉 | **信号完全弥散**——需全部 896 维才能全翻转，无热点维度 | M7-B, M7-G |
| 🏅 | **生成阶段不可控**——prefill 向量在自回归生成中反向增加谄媚 | M7-F |
| 🏅 | **Hallucination ≠ Sycophancy**——前者有方向性子空间，后者没有 | A3, S, X2 |

## 五、战略分歧点（更新）

### 分歧 1：挖掘 vs 注入 —— **已解决**

| 路径 | 做法 | 结论 |
|---|---|---|
| **挖掘**（M7-A/B/C/D/F/G） | 在现有权重空间中寻找因果信号 | 信号存在但完全弥散，无法定位到少数单元 |
| **注入**（M7-H 待执行） | 通过微调强制压缩出独立回路 | **现在是唯一可行路径** |

**结论**：挖掘阶段已穷尽——信号弥散在全896维中，无热点。必须通过注入（LoRA微调）创造独立的控制通道。

### 分歧 2：Attention vs MLP —— **已解决**

| 路径 | 依据 | 结论 |
|---|---|---|
| **Attention** | 外部 LLM #1/#4 分析 | 效果弱于 MLP |
| **MLP** | M7-A 数据 | **MLP 是主要信号载体** |

**M7-G 新证据**：PCA 子空间分析表明信号同时穿透 MLP 和 Attention（REPLACE 全残差流才达100%），但 MLP 贡献更大。

### 分歧 3：方向加法 vs 状态替换 vs 维度扩展 —— **部分解决**

| 方法 | 效果 | 根本原因 |
|---|---|---|
| 方向加法（steering） | ❌ 无效 | 保留类内方差（M7-C） |
| 状态替换（REPLACE） | ✅ 100%（prefill）| 消除所有方差 |
| PCA 子空间（M7-G） | ⚠️ 部分有效 | K=20→Δ=-0.30，需 K=896→Δ=-0.55 |
| LoRA adapter（M7-H） | ❓ 待验证 | 理论可行：学去噪映射 hs_syc→hs_nonsyc |

**关键洞察**：谄媚翻转是**方差坍塌**问题，不是**均值偏移**问题。任何只做加法不做替换的方法都会失败。
